# SDL-Snake 

## Build and Run

Make sure `sdl2` and `sdl2_ttf` are installed.

```
git clone https://github.com/zack466/SDL-Snake
cd SDL-Snake
cmake .
make run
```

## Features

- It's just snake, I'm not sure what you expect
- Have fun!

## Controls

- `wasd` or arrow keys to control the snake
- `q` to quit.
